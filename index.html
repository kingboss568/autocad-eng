Use the verified file listed in R2_DELIVERY.md. Keep the bucket private and generate signed URLs or time-limited download URLs.

2. Connect Gumroad sale event

Use Gumroad Resource Subscriptions or a Ping-style webhook so a sale event reaches a Cloudflare Worker endpoint.

3. Worker creates the link

The Worker verifies the sale, creates a private R2 download URL, stores a fulfillment record, and sends an email with the link.

4. Buyer receives English instructions

The email should include product name, download expiry, support email, file size, and the SHA-256 checksum.
