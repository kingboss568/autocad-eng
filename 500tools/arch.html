Architecture workflow tools.

Use these English prompts and checklists with the CAD bundle when assembling plans, elevations, sections, details, and drawing sets.

CAD QAFaster checks before sending a drawing package.

Sheet Setup Checklist

  • Confirm title block, scale, north arrow, and revision date.
  • Check file naming against the project issue date.
  • Keep model space and paper space standards consistent.

Detail Review Prompt

  • Does the detail show material changes clearly?
  • Are dimensions readable at the printed scale?
  • Are references coordinated with the plan callout?

Annotation Cleanup

  • Remove duplicate notes and stale leader text.
  • Use consistent abbreviations and text heights.
  • Verify all callouts point to visible elements.

Drawing Issue List

  • Missing dimensions.
  • Layer visibility conflicts.
  • Unresolved external references.
  • Uncoordinated room names or detail numbers.

CAD Block Intake

  • Audit insertion point and unit scale.
  • Move blocks onto project layers.
  • Rename reusable blocks before issuing files.

Handoff Note

This page supports production review only. Final drawings still need project-specific professional review, code review, and engineering coordination.
